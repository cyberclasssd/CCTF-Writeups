# Metaphysics
* **Category:** Misc
* **Points:** 100
## Problem
Metaphysics is a shadow cast on the wall of a cave. Can you find the flag?
## Solution
Using a tool like GIMP, we can view the metadata. The flag is contained in the artist's name in the EXIF data.
